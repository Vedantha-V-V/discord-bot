# Discord Bot
---

Discord Bot that provides details about upcoming events and also capabilities to perform multiple real time CRUD operations.

**Tech Stack & Tools:**
- NodeJS
- MongoDB
- Disord Developer Portal

**Components & Layout:**
- [x]  interaction commands like /help and /hello
- [ ]  Tabular formatted message for better readability 
- [ ]  Use clear typography

**Functionalities:**
- [x]  Fetch all upcoming events
- [x]  Add a new event
- [x]  Update an existing event
- [x]  Delete previously occured events
